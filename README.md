## Features

- Change 4 border-radius values and see changes.
- Copy to clipboard the resulting css.

- ⚛️ **React Js** — A JavaScript library for building user interfaces

## Getting started

- Clone project > enter the project folder
- run `yarn`
- run `yarn start`
